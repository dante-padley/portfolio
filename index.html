Sndr Rcvr

Sndr Rcvr is a proof-of-concept project meant to use inter-process communication (IPC) to send a message between two running processes on the same system. The sndr process creates a message queue in a shared memory space. The rcvr process attaches to that same memory space and ingests the message. It then sends a message back to sndr to confirm that it got the message.
